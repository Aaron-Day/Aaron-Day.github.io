Senior Project: AMMS

Aviation Maintenance Management System. Solo Software Engineering Program Senior Project. A web-based version of the army's Unit Level Logistic Software - Aviation (Extended) [ULLS-A(E)]. This project utilizes ASP.NET Core 2.0 multi-platform development framework as well as Microsoft SQL Server to construct a full-stack aviation maintenance management program.
